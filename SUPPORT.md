# Support

## Getting Help

Thank you for using PeyDot! Here's how you can get help:

### Documentation

- **Main Docs**: [docs.peydot.app](https://docs.peydot.app)
- **GitHub Wiki**: Check our [wiki](https://github.com/peydot/peydot-magic-links/wiki)
- **FAQ**: See our [FAQ section](./docs/FAQ.md)

### Community

- **Discord**: Join our [Discord community](https://discord.gg/peydot) for real-time support
- **Twitter**: Follow us [@peydotapp](https://twitter.com/peydotapp)
- **Telegram**: Join our [Telegram group](https://t.me/peydot)

### Issue Tracking

If you find a bug or have a feature request:

1. **Search existing issues** first
2. **Create a new issue** with:
   - Clear title
   - Detailed description
   - Steps to reproduce (for bugs)
   - Environment details

### Reporting Security Vulnerabilities

For security issues, **DO NOT** create a public issue. Instead:

- Email: security@peydot.app
- Or use GitHub's private vulnerability reporting

We appreciate your responsible disclosure.
