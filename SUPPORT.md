## 🐛 Bug Reports

Found a bug? Please report it by:
1. Searching existing issues
2. Creating a new issue with:
   - Clear description
   - Steps to reproduce
   - Environment details
   - Screenshots if applicable

## 💡 Feature Requests

Have a feature idea? We'd love to hear it!
1. Check our roadmap for existing plans
2. Open a GitHub issue with your suggestion
3. Explain the use case and benefits
